Weather Project

Project Instructions

● collect weather information (temp, pressure, wind) from the
METAR service for a variety of airports (python container, you
can use the python code from the exercise)

● store this information in a persistent database (database
container)

● from a Web frontend, the user can select one of the airports and
the stored results will be presented

● You should use (at least) three containers: the “collector”, the
database, the frontend

● frontend TLS secured
